# 4 проект курса

## О проекте
Этот репозиторий является результатом верстки сайта магазина кофе по макету. На данный момент сделана структура сайта и его стилизация.

Используются технологии HTML5 и CSS. Для стилей используется методология BEM; выбрана она, так как она мне показалась самой интуитивной: по названию сразу понятно какой стиль относится к какому элементу. ACSS мне показался слишком громоздким по использованию и по сути переносящим спецификацию CSS в DOM, а OOCSS не выбран примерно по тем же причинам что и ACSS.
