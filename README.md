# 4 проект курса

## О проекте
Этот репозиторий является результатом верстки сайта магазина кофе по макету.

На данный момент сделана структура сайта и его стилизация, а также адаптация к разным размерам экранов, например мобильных.

Используются технологии HTML5 и CSS.

Для стилей используется методология BEM; выбрана она, так как она мне показалась самой интуитивной: по названию сразу понятно какой стиль относится к какому элементу. ACSS мне показался слишком громоздким по использованию и по сути переносящим спецификацию CSS в DOM, а OOCSS не выбран примерно по тем же причинам что и ACSS.

Адаптивная верстка выполнена в порядке desktop-first.

Используемые брейкпоинты по ширине:
* 1080px
* 900px
* 720px
* 640px
* 480px
* 380px
* 360px

Используемые брейкпоинты по высоте:
* 720px
